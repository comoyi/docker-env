# docker environment

## Setup step

1.clone

```bash
git clone git@github.com:comoyi/docker-env.git
```

2.copy .env.example to .env

```bash
cp .env.example .env
```

3.run

```bash
docker-compose up
```

open browser and visit

## Usage

build

```bash
docker-compose build
```

up

```bash
docker-compose up
```
